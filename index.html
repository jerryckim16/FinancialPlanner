Investments

Initial Position is how much you currently hold in each investment. Monthly Contribution % controls how your surplus (after costs, loans, and cash savings split) flows into each investment — percentages should sum to 100%. Return is annual capital appreciation. Dividend is a bonus annual yield paid to savings as cash (on top of Return — total return = Return + Dividend). Expense Ratio is subtracted from Return before compounding.
